# Hugo theme

## Getting started

### GNU Make

**Windows**

```console
scoop install make
```

**Linux**

```console
sudo apt install make
```

**MacOS**

```console
brew install make
```

### Hugo

Install [hugo-extended](https://gohugo.io/getting-started/installing/)

## Development

```console
make run
make build
```
