# Hugo theme

## Getting started

### GNU Make

Windows

```bash
winget install -e GNU.Make
```

Linux

```bash
sudo apt install make
```

### Hugo

Download `hugo extended`: https://github.com/gohugoio/hugo/releases

## Development

```bash
make run
make build
make docker
```
